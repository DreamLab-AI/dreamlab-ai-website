## How to Use This Documentation

### For New Visitors

1. Start with [Website Guide](./WEBSITE_GUIDE.md) to learn how to navigate
2. Browse [Workshops](../../workshops) for training options
3. Visit [FAQ](./FAQ.md) for common questions
4. Use [Contact Form](../../contact) to get in touch

### For Workshop Booking

1. Go to [Workshops](../../workshops) page on the website
2. Review workshop details
3. Use [Workshop Booking Guide](./WORKSHOP_BOOKING.md) for step-by-step help
4. Submit inquiry through the Contact form

### For Residential Training

1. Visit [Residential Training](../../residential-training) page
2. Review programs and curriculum
3. Check [FAQ](./FAQ.md) for residential program questions
4. Submit booking inquiry

### For Sovereign AI Systems Training

1. Read [Nostr Setup Guide](./NOSTR_SETUP.md) to prepare
2. Create your Nostr identity
3. Understand the protocol before training
4. Ask questions during your program

### For Any Questions

- Check [FAQ](./FAQ.md) first - it covers most topics
- Use the [Contact Form](../../contact) to reach out
- Browse [Team](../../team) page to find specific instructors
- Email us after your initial inquiry
