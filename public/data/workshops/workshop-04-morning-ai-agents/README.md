# Workshop 04 - Morning Session: AI Agents

## Overview
Master the art of building intelligent AI agents that can reason, use tools, and solve complex problems autonomously. This hands-on workshop covers the production-grade tools and patterns for creating AI agents in 2026 -- from the Claude API and MCP protocol to Claude Code workflows and the Anthropic Agent SDK.

## What You'll Build
- **Tool-Using Agent**: Claude API agent with structured tool calling
- **Claude Code Workflow**: Multi-step coding agent with CLAUDE.md configuration
- **Research Agent**: Autonomous web research with source tracking and citations
- **Multi-Agent Pipeline**: Coordinated specialist agents collaborating on a task

## Key Technologies (2026)
- **Claude API** - Anthropic's tool use with Sonnet 4.6 / Opus 4.8
- **Claude Code** - Terminal coding agent with subagents and workflows
- **MCP (Model Context Protocol)** - Standardised tool integration
- **Anthropic Agent SDK** - Python/TypeScript SDK for custom agents
- **OpenAI Codex CLI** - Open-source terminal agent
- **Aider** - Open-source multi-model pair programming
- **Cursor / Windsurf** - IDE-integrated AI agents
- **LangChain / LangGraph** - Framework-agnostic orchestration

## Session Structure
- **Duration**: 3 hours
- **Format**: Live coding + hands-on exercises
- **Difficulty**: Intermediate to Advanced
- **Prerequisites**: Python or TypeScript, API basics, LLM fundamentals
