# Module 08: Vibe Coding Mastery

## Overview
This module covers Vibe Coding Mastery as part of the AI-Powered Knowledge Work workshop. Vibe coding is the practice of building software by describing what you want in plain English and letting an AI coding agent handle the implementation. Updated for mid-2026, this module covers the current tool landscape including Claude Code, Cursor AI, Windsurf, GitHub Copilot, and more. No programming experience is required — the emphasis is on clear communication, iterative refinement, and creative direction.
