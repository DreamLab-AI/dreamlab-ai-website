# Chapter 2: Getting Started: Setup and Initial Configuration

Successfully leveraging AI coding agents begins with straightforward setup. This chapter covers getting started with the two primary terminal agents — Claude Code and OpenAI Codex CLI — as well as web-based and IDE-integrated alternatives.

We will cover:

*   [Web & IDE Tools](./02_a_accessing_the_codex_cloud_agent.md): Getting started with ChatGPT, Claude.ai, Cursor, and GitHub Copilot for coding.
*   [Setting Up Terminal Agents](./02_b_setting_up_the_codex_cli.md): Installation and configuration for Claude Code CLI and OpenAI Codex CLI.
*   [API Keys and Authentication](./02_c_api_keys_and_authentication.md): Obtaining, handling, and using API keys securely across providers.

Proper setup is fundamental to unlocking the powerful features offered by these tools. Most developers will want at least one terminal agent (Claude Code or Codex CLI) and one IDE-integrated tool (Copilot or Cursor) for a complete workflow.
