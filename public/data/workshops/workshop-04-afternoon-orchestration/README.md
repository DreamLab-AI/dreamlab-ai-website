# Workshop 04 - Afternoon Session: Agent Orchestration & Safety

## Overview
Move beyond single agents to production multi-agent systems. This session covers orchestration patterns, safety guardrails, cost management, and real-world deployment strategies using Claude Code workflows, MCP servers, and the Anthropic Agent SDK.

## What You'll Build
- **Subagent Pipeline**: Parent agent delegating to specialist child agents
- **Parallel Research System**: Multiple agents searching and synthesising concurrently
- **Guardrailed Agent**: Production agent with cost caps, timeouts, and human-in-the-loop approval
- **Full Orchestration Project**: End-to-end multi-agent system with monitoring and safety controls
