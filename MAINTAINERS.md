DreamLab AI Website is maintained by a small group with commit access, working in the open.
Decisions are recorded in issues and PRs.

## Upstream

This project's community forum is powered by [nostr-rust-forum](https://github.com/DreamLab-AI/nostr-rust-forum),
a Rust kit built on [solid-pod-rs](https://github.com/DreamLab-AI/solid-pod-rs), which is itself
a Rust port of Melvin Carvalho's [JavaScriptSolidServer (JSS)](https://github.com/JavaScriptSolidServer/JavaScriptSolidServer).
JSS is the reference implementation of the Solid Protocol and the canonical source for the
feature set, protocol extensions, DID:Nostr identity method, and Web Ledger micropayment system
consumed by this ecosystem.

See [.github/CODEOWNERS](.github/CODEOWNERS) for path-level review routing.

## Process

Maintainers follow the same workflow as other contributors (issue → branch → PR → review → merge).

## Becoming a maintainer

By invitation of an existing maintainer, after demonstrated substantive
contribution. No formal vote; existing maintainers make the call and
update this file.
