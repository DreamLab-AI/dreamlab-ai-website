# Workshop 04 - Morning Session: AI Agents

## Overview
Master the art of building intelligent AI agents that can reason, use tools, and solve complex problems autonomously. This hands-on workshop covers the latest frameworks and patterns for creating production-ready AI agents in 2025.

## What You'll Build
- **Research Agent**: Autonomous web research with citations
- **Code Assistant**: AI that writes, tests, and debugs code
- **Data Analysis Agent**: SQL generation and visualization
- **Multi-Agent System**: Coordinated agents solving complex tasks
