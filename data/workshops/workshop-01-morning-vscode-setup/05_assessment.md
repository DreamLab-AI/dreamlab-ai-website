# Chapter 5: Assessment - Validating Your Mastery

## Introduction: Proving Your New Skills

This assessment validates that you've mastered VS Code as your AI command centre. It combines knowledge checks with practical demonstrations.

**Time:** 10 minutes
**Passing Score:** 80% (24/30 points)
**Format:** Multiple choice, practical tasks, and reflection

## Part 1: Knowledge Check (10 points)

### Question 1: Core Concepts (2 points)

What is the primary advantage of using VS Code with direct AI integration over web-based AI tools?

A) It's faster to load
B) It maintains project context and allows batch processing
C) It has better graphics
D) It's free

**Answer:** B

---

### Question 2: Extensions (2 points)

Which extension type is NOT typically needed for an AI-enhanced document workflow?

A) Markdown support
B) AI assistant (Continue, Copilot)
C) Game development tools
D) Version control (GitLens)

**Answer:** C

---

### Question 3: AI Integration (2 points)

What's the keyboard shortcut to invoke AI assistance in Continue?

A) `Ctrl+A`
B) `Ctrl+I`
C) `Ctrl+P`
D) `Ctrl+Z`

**Answer:** B

---

### Question 4: Project Structure (2 points)

Why is organizing files into folders important in VS Code?

A) It looks professional
B) It helps AI understand context and improves suggestions
C) It's required by VS Code
D) It saves disk space

**Answer:** B

---

### Question 5: Workflow Efficiency (2 points)

What's the most efficient way to create repetitive document structures?

A) Copy-paste from old files
B) Type everything manually
C) Use snippets with AI enhancement
D) Download templates from the internet

**Answer:** C
